# Repo short texts

## One-line description
A hard-delivery runtime and validation kit for OpenClaw report workflows.

## Short pitch
openclaw-hard-delivery packages a production-oriented delivery pattern for OpenClaw: send, enter pending confirmation, observe real delivery, reconcile from evidence, and clean up into terminal success.

## Two-sentence pitch
This repository turns a real OpenClaw hardening effort into a reusable delivery runtime and validation kit. It is designed for workflows where `sessions_send` timeouts can still coexist with actual message delivery, so the system must close from observation and reconciliation instead of naive timeout-as-failure logic.
